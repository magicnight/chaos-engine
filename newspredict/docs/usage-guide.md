## 产品简介

NewsPredict 是一个 OSINT（开源情报）驱动的预测市场平台。它从 C.H.A.O.S. Engine 获取实时情报数据（涵盖 46 个数据源），自动生成预测市场，用户可以对真实世界事件的走向进行交易。

**核心理念**：用市场机制聚合群体智慧，让信息价格化。

**主要特性**：
- 实时 OSINT 数据驱动的市场生成
- LMSR（对数市场评分规则）动态定价
- 支持虚拟额度交易和 BSC 链上交易
- 自动解决到期市场
- 中英文双语界面

## 快速开始

### 1. 登录

打开 NewsPredict 首页，点击登录按钮。支持以下登录方式：

| 方式 | 说明 |
|------|------|
| **访客登录** | 一键开始，获得 $1,000 虚拟额度，无需注册 |
| **Google 登录** | 使用 Google 账号授权登录 |
| **GitHub 登录** | 使用 GitHub 账号授权登录 |
| **钱包登录** | 连接以太坊/BSC 钱包（MetaMask 等），通过 SIWE 签名验证 |

> 推荐新用户使用**访客登录**快速体验，之后可绑定社交账号或钱包。

### 2. 浏览首页

登录后进入首页，你将看到：

- **头条市场**：当前最受关注的预测事件
- **市场卡片**：多个活跃市场的缩略展示
- **趋势区域**：正在上涨或交易活跃的市场
- **快速投票**：一键参与热门预测
- **突发情报**：来自 CHAOS Engine 的最新 OSINT 事件
- **已结算市场**：最近解决的预测结果

### 3. 进行第一笔交易

1. 点击任意市场卡片进入详情页
2. 查看当前 YES/NO 价格和价格走势图
3. 选择你的立场（**YES** 或 **NO**）
4. 输入交易金额
5. 点击 **买入** 按钮完成交易

## 核心功能

### 浏览市场

#### 首页（Home）

首页展示精选内容，包括头条市场、趋势市场、快速投票和突发情报。数据实时更新，通过 SSE（Server-Sent Events）与 CHAOS Engine 保持同步。

#### 探索页（Explore）

探索页提供完整的市场列表，支持：

- **搜索**：输入关键词搜索市场问题
- **分类筛选**：全部 / 市场 / 政治 / 科技 / 冲突 / 气候 / 健康 / 网络安全 / 太空
- **实时刷新**：每 30 秒自动更新数据

#### 市场列表（Markets）

市场列表页展示所有开放市场的详细信息，包括问题、分类、YES/NO 价格、交易量等。点击 **+ 创建** 可以创建新市场。

### 交易操作

#### 市场详情页

每个市场的详情页包含：

- **价格展示**：当前 YES 和 NO 的实时价格
- **价格走势图**：历史交易形成的价格曲线
- **交易面板**：执行买入操作
- **市场统计**：交易量、参与者数量、截止日期
- **解决标准**：明确的判定规则和数据来源
- **最近动态**：其他用户的交易记录

#### 如何交易

1. **选择方向**：点击 **BUY YES**（看涨）或 **BUY NO**（看跌）
2. **设定金额**：手动输入或快捷选择 $5 / $10 / $25 / $50 / $100
3. **查看详情**：
   - 预计份额：你将获得的份额数量
   - 均价：当前成交价格
   - 预期收益：如果预测正确的潜在收益
4. **确认交易**：点击交易按钮完成

#### 价格含义

- 价格 = 该结果发生的市场预期概率
- YES 价格 $0.70 = 市场认为有 70% 概率发生
- NO 价格 $0.30 = 市场认为有 30% 概率不发生
- YES + NO 价格始终等于 $1.00

#### 盈利方式

- 如果你买了 YES 且结果为 YES → 每份额获得 $1.00
- 如果你买了 YES 且结果为 NO → 份额价值归零
- **利润 = 结算价值 - 购买成本**

### 创建市场

登录后可以创建自定义预测市场：

1. 进入 **创建** 页面（底部导航栏 "+" 按钮）
2. 填写以下字段：

| 字段 | 必填 | 说明 |
|------|:----:|------|
| 问题 | * | 以"是否"形式表述的预测问题（最多 500 字） |
| 描述 | | 为市场提供更多背景信息（最多 2000 字） |
| 分类 | * | 选择市场所属分类 |
| 截止日期 | * | 市场关闭时间 |
| 解决标准 | * | 明确的判定规则（最多 1000 字） |
| 数据来源 | | 判定结果的参考来源（如 CoinGecko、Reuters） |

3. 点击 **创建市场** 按钮

> 好的预测问题示例：
> - "BTC 在 2026 年 4 月 30 日前会突破 $100,000 吗？"
> - "WHO 会在 2026 年 6 月前宣布新的全球卫生紧急事件吗？"

### 投资组合

投资组合页面展示你的交易全貌：

- **余额**：当前可用虚拟额度
- **未实现盈亏**：持仓的浮动盈亏
- **活跃持仓**：当前所有开放市场中的持仓
- **已结算持仓**：已经解决的市场持仓及盈亏结果
- **统计概览**：盈亏总额、活跃持仓数、胜率

### 排行榜

排行榜展示平台上的顶级交易者：

- **今日榜**：过去 24 小时表现最佳的交易者
- **本周榜**：过去 7 天的排名
- **总榜**：历史累计表现排名

排名依据包括：盈亏金额、交易次数、胜率。

### 动态中心

动态页面记录你的所有交易活动：

- **交易记录**：买入的方向、份额数量、成交价格
- **市场结算**：预测正确/错误的结果和盈亏金额

点击任意动态可跳转到对应市场详情。

## 交易模式

### 虚拟额度模式

默认交易模式，使用平台内虚拟美元进行交易。

- 新用户注册获得 **$1,000** 初始额度
- 交易范围：$1 ~ $10,000
- 无需加密钱包
- 适合学习和体验预测市场

### 链上交易模式

连接钱包后可切换到链上模式，使用 C.H.A.O.S. (CRUX) 代币在 BSC（BNB Smart Chain）上进行链上交易。

**准备工作**：
1. 安装 MetaMask 或其他兼容钱包
2. 添加 BSC 网络
3. 确保持有 CRUX 代币和少量 BNB（用于 Gas 费）

**链上交易流程**：
1. 连接钱包
2. 切换到 **链上 (CRUX)** 模式
3. 点击 **授权 CRUX** — 批准智能合约使用你的代币
4. 等待授权交易确认
5. 点击 **买入份额** — 执行预测交易
6. 等待交易确认

**合约地址（BSC 主网）**：
- CRUX 代币：`0xcE3fbb08D72BEd7F645F59FE0f031659b5B298c4`
- 预测市场：`0xAa7208Cf64078756fB58698fbE748DC3c9b4Cb88`

> 链上交易使用 LMSR 算法在智能合约中计算价格，确保公平透明。

## 定价机制

NewsPredict 使用 **LMSR（对数市场评分规则）** 进行自动做市。

### 工作原理

- 系统作为自动做市商，始终为 YES 和 NO 双方提供流动性
- 价格由 sigmoid 函数决定：`P(YES) = 1 / (1 + e^(-(yes_shares - no_shares) / b))`
- 参数 `b`（流动性参数，默认 100）控制价格对交易的敏感度
- 每笔交易都会改变价格 — 买入越多，价格越高

### 关键特性

| 特性 | 说明 |
|------|------|
| 始终有流动性 | 无需等待对手方，随时可以交易 |
| 价格 = 概率 | YES 价格直接反映市场预期概率 |
| 越早越便宜 | 在价格低时买入，如果最终正确，获利更大 |
| 有限亏损 | 最大亏损不超过你的投入金额 |

## 市场生命周期

```
创建 → 开放交易 → 到期关闭 → 数据验证 → 结算 → 派发收益
```

1. **创建**：由 CHAOS Engine 自动生成或用户手动创建
2. **开放交易**：用户自由买卖 YES/NO 份额
3. **到期关闭**：到达截止日期后，停止交易
4. **数据验证**：根据解决标准和数据来源判定结果
5. **结算**：系统根据结果确定 YES 或 NO 获胜
6. **派发收益**：获胜方按持仓份额获得收益

## 常见问题

### Q: 我的虚拟额度用完了怎么办？

目前没有充值机制。你可以通过成功的预测交易来增加余额，或重新以访客身份登录获取新的 $1,000 额度。

### Q: 市场是谁创建的？

市场有两种来源：
- **系统市场**：由 CHAOS Engine 根据实时 OSINT 数据自动生成
- **用户市场**：由已登录用户手动创建

### Q: 如何判定市场结果？

每个市场都有明确的"解决标准"和"数据来源"。到期后，系统会根据这些标准参照真实世界数据进行判定。

### Q: 链上交易和虚拟交易有什么区别？

| | 虚拟交易 | 链上交易 |
|---|---------|---------|
| 货币 | 平台虚拟美元 | CRUX 代币 (BEP-20) |
| 需要钱包 | 不需要 | 需要 MetaMask 等 |
| Gas 费 | 无 | 需要少量 BNB |
| 透明度 | 平台数据库 | BSC 区块链公开可查 |
| 适合 | 学习体验 | 真实价值交易 |

### Q: 价格是如何变动的？

价格由 LMSR 自动做市算法决定。当更多人买入 YES 时，YES 价格上升，NO 价格下降；反之亦然。价格始终在 $0 ~ $1 之间。

### Q: 可以卖出已持有的份额吗？

目前支持买入操作。卖出功能（链上）已在智能合约中实现，前端卖出界面将在后续版本中推出。
